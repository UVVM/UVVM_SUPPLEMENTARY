- bitvis_vip_rgmii  : A VIP that uses UVVM
  - doc             : Quick Reference for BFM and VVC
  - script          : Compile scripts
  - src             : RGMII BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for RGMII.
The given VIP complies with the basic RGMII protocol and thus allows a normal 
access towards a RGMII interface. This VIP is not a RGMII protocol checker.
