-----------------------------------------------------------
-- Directory structure                                   -- 
-----------------------------------------------------------

- bitvis_vip_axi        : A VIP that uses UVVM
  - doc                 : Quick Reference for BFM and VVC
  - script              : Compile scripts
  - src                 : AXI4 VIP BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for AXI4.
The given VIP complies with the AXI4 protocol and thus allows a normal 
access towards a AXI4 interface. This VIP is not a AXI4 protocol checker.
