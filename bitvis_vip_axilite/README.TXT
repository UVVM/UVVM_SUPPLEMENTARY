-----------------------------------------------------------
-- Directory structure                                   -- 
-----------------------------------------------------------

- bitvis_vip_axilite    : A VIP that uses UVVM
  - doc                 : Quick Reference for BFM and VVC
  - script              : Compile scripts
  - src                 : AXI4-Lite VIP BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for AXI4-Lite.
The given VIP complies with the basic AXI4-Lite protocol and thus allows a normal 
access towards a AXI4-Lite interface. This VIP is not a AXI4-Lite protocol checker.
