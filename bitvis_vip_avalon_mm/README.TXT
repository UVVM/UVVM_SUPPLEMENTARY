- bitvis_vip_avalon_mm      : A VIP that uses UVVM
  - doc               	    : Quick Reference for BFM and VVC
  - script            	    : Compile scripts

IMPORTANT
This is a simplified Verification IP (VIP) for Avalon MM.
The given VIP complies with the basic Avalon MM protocol and thus allows a normal 
access towards a Avalon MM interface. This VIP is not a Avalon MM protocol checker.
