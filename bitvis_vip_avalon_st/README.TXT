This VIP can be used as master and slave.

- bitvis_vip_avalon_st      : A VIP that uses UVVM
  - doc               	    : Quick Reference for BFM and VVC
  - script            	    : Compile scripts
  - src               	    : Avalon ST VIP BFM and VVC source code

IMPORTANT
This is a simplified Verification IP (VIP) for Avalon ST.
The given VIP complies with the basic Avalon ST protocol and thus allows a normal 
access towards a Avalon ST interface. This VIP is not a Avalon ST protocol checker.
