- bitvis_vip_wishbone   : A VIP that uses UVVM
  - doc               	: Quick Reference for BFM and VVC

IMPORTANT
This is a simplified Verification IP (VIP) for Wishbone.
The given VIP complies with the basic Wishbone protocol and thus allows a normal 
access towards a Wishbone interface. This VIP is not a Wishbone protocol checker.
